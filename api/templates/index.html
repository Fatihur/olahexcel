Aplikasi Pengolah Data Biaya Perjalanan

aplikasi ini fatih buat bantu ayang buat laporan biaya perjalanan. Aplikasi ini otomatis ngerubah file word ke excel

Cara Pakai:

  1. Upload file Word-nya yang udah dikasi keterangan ya❤️
  2. Pilih proses, dan tunggu sebentar... ⏳
  3. Simpan file hasilnya, terus buka 📂
  4. Kalau ada peringatan, klik aja "Yes" 💕
  5. Pilih "Enable Editing" yaa 😘

Aplikasi ini belum sempurna, masih banyak kekurangan... kalau tabelnya kurang rapi, rapiin aja yaa.. semangat terus, cantik!
